Projects

GovWatch

This was my first project. It utilizes the Congress gem with data from the Sunglight Foundation. Users can see recent activity in both houses of Congress, as well as see who their representatives are. This was built using HTML, CSS, and Ruby on Rails.

URL: http://govwatch.herokuapp.com

Vibe

This was the second project and was a group project. For this project I worked on the front-end. I was primarily reponsible for the jQuery animations. Enter in a city and you will see a set of images from different social media sites animate across the page. This will give you the "vibe" in that location.

URL: http://thevibe.herokuapp.com

PetSitter

The idea for my third project came when taking a weekend trip. I needed someone to watch the cat while away. So I decided to make the PetSitter app. Sitters can see what needs to be done and when. Pet guardians receive a message letting them know when tasks have been completed and their pet is being taken care of. This app uses Ruby on Rails in the back-end and jQuery mobile on the front-end. This app isn't yet finished. The screenshots are concepts, I'm still finalizing the UI.

URL: Still under construction.
